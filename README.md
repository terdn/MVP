# ERDN COSMETICS

A premium React Native Expo app for personalized skincare and makeup color recommendations.

## Features

- **Welcome Screen**: Clean branding with "ERDN COSMETICS" and tagline "You don't try, you know."
- **Subscription Plans**: 
  - Standard: Skincare recommendations only
  - Premium: Skincare + Makeup color suggestions with 48h free trial
- **Camera Screen**: Face photo capture for analysis
- **Analysis Screen**: Personalized recommendations (photo deleted immediately after analysis)
- **Black & White Minimal Design**: Premium, clean UI

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Run on iOS:
```bash
npm run ios
```

4. Run on Android:
```bash
npm run android
```

## Important Notes

- **No Medical Claims**: All recommendations are for cosmetic purposes only. Not a medical service.
- **Privacy**: Photos are deleted immediately after analysis.
- **Camera Permissions**: The app requires camera permissions to capture face photos.

## Technology Stack

- React Native
- Expo SDK 50
- React Navigation
- Expo Camera
- Expo File System
